# AVIATE TASK
### Anirudh Mishra

Built using React.js and Django Rest Framework.

***Deployed Link:*** [https://anirudhmishra-aviate-task.netlify.app/candidates/1](https://anirudhmishra-aviate-task.netlify.app/candidates/1)

***API Link:*** [https://anirudhgray.pythonanywhere.com/](https://anirudhgray.pythonanywhere.com/)

How to run frontend locally:
```bash
cd frontend
npm i
npm run start
```

How to run api locally:
```bash
cd api
pip install -r requirements.txt
python manage.py migrate candidates
python manage.py runserver
```
